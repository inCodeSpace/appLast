Встречи

- - - - - - - -
Yii2 MySQL JQuery AJAX

Описание проекта:

Сайт организации групп и встреч, для 4х стран по регионам и городам.
• Выполнен на Yii2 Advanced
• Существующий проект на Yii1, с нуля (кроме частино-css) создан на Yii2
• Использовался GitLab
• Создание групп с заданием интересов
• Создание встреч для данной группы
• Альбомы с фотографиями по группам/встречам
• Мультиязычность...

Подробно ▼ ▲

Меню:

- 2х уровневые пункты меню (данные из БД для данного пользователя)
- Yii2 bootstrap Nav, NavBar

StepWizard c ActiveForm:

- StepWizard состоящий из (AJAX) связанных выпадающих списков с автокомплитом
(kartik-DepDrop\Select2)
- Данные в списках из БД, обычные поля и поля с возможностью добавлением ключевых слов.

Стена (комментарии):

- Стена: 2-х уровневых комментариев с подгрузкой новых комментариев по AJAX-у
(полученные данные в формате JSON используются для генерации HTML-я).
AJAX-добавление нового комментария (данные заносятся в БД) комментарии выводятся
в целевое место (либо в начало стены, либо ниже в блок комментария, если это ответ на
комментарий и выполняется прокрутка к место добавленного комментария).

Разное:

- PJAX Grid-ы с пагинацией встроенные в страницы.
- Создание нового пользователя, с отправкой ссылки на почту с подтверждением.
- Модальные окна с AJAX формами или сборными формами (формы с скрытыми полями) с использованием TimePicker, DatePicker, модальное окно по успешному завершению.
- Встроенный виджет Pjax ListView с пагинацией и свитчером отображения (видом и данных).
- Спойлеры.
- Модальное окно входа на сайт (с авторизацией через соц. сети (сделано не до конца)).
- JS-Календарь: ячейки дней заполняются данными из БД
(в виде ссылок на данное событие, route).
- Отправка пригласительного письма по почте.
- Фотоальбом и фотогалерея.
- Создание нового фотоальбома и загрузки фотографий с созданием миниатюр.

pickupMyNow

- - - - - - - -
Yii2 webSockets Ratchet ZeroMQ

Описание проекта:

Приложение помогающее добраться с точки А до точки Б
• Пересылка координат между клиентом и клиентом через сервер с помощью webSocket-ов
• Чат на webSocket-ах (в браузере)
• Авторизацию/аутентификацию через FaceBook с использованием API budgettaxi
• Небольшой личный кабинет с загрузкой изображений пользователем.

Подробно ▼ ▲

- Авторизация/аутентификация частично обработка route, обработка данных связанных с renew_token и тп.
- Интерфейс "документов" пользователя (загрузка изображений и их Yii2 CRUD)

Ratchet:

- Ratchet: (Библиотека php для работы с webSockets, socketo.me)
- Разбирался как с ней работать и интегрировать её в Yii2.
- Разбирался с работой Ratchet PUSH (отправка данных клиентам с сервера).
- Описывание интрефейсов Ratchet, реализация основных (не всё) методов Ratchet
- Интеграция ZeroMQ (http://zeromq.org/) (работа с web socket-ами обслуживание очередей)
- п.1 Отправка сообщений клиент-сервер-клиент(ы) (через консоль)
- п.2 Отправка сообщений клиент-сервер-клиент(ы) (браузер js-script)
- Чат в браузере: (на базе п.1 и п.2 и как html-ответ на реализованные
интерфейсы Ratchet) прием и обработка обращений из webSocket
(по сути разбор JSON туда-сюда и вывод его в чат на jQuery)

Что я не делал в проекте:

- Личный кабинет пользователя
- Установка и настройка расширения для работы с ролями
- auth. from facebook API
- Поднят WebSocket server
- Доступна в браузере google-карта для конкретного клиента с просмотром пути и текущих координат.
- Используется алгоритм апроксимации для пересылания координат.

▲

Технологии:

Yii2 Advanced, webSocket, Ratchet, ZeroMQ, MySQL, Facebook API, budgettaxi API, jQuery

mailSender

- - - - - - - -
Yii2 CRUD

Описание проекта:

Создание почтовых рассылок (по CRON) и их отправка на email.
• CRUD данных email-ов и рассылок (с орг. очередей их рассылки).
• В Yii2 реализован вход (форма) для пользователя
• Имеется форма для добавления email-ов (в DB) либо возможность загрузить файл с определенной
структурой записи данных email-ов
• Форма создания рассылки заносящаяся в таблицу очередей
• По CRON-у выполняется периодическая отправка почты.
• Логирование выполнения рассылки
• Имеется CRUD рассылок, e-mail-ов, очередей.

Технологии:

Yii2, MySQL.
